Project Title: Password Strength Analyzer with Custom Wordlist Generator

Description:
This tool analyzes password strength using zxcvbn and generates a
custom wordlist based on user inputs like name, birth year, and pet name.

Tools Used:
- Python
- zxcvbn library

Features:
- Password strength scoring
- Custom wordlist generation
- Wordlist export in .txt format

Ethical Use:
This project is for educational and ethical testing purposes only.
